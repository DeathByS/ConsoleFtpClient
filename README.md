# ConsoleFtpClient
ConsoleFtpClient

# 소개
c, winsock을 사용하여 콘솔 FTP 클라이언트 제작

파일의 다운로드 부분 제작.

1. 기존의 ftp 클라이언트와 유사하게 제작
2. 서버 접속이 완료된 소켓과 다운로드 할 파일 이름을 인자값으로 받음
3. 다운도르를 위해 passive mode로 변경
4. 바이너리 모드로 변경
5. 파일을 생성 후, 다운로드 받은 내용을 파일에 기록


# 사용 기술 및 개발 툴
* C
* WinSock
* Visual Studio
